Modification d'équipe

Comment ajouter, copier et transférer des joueurs

Ici vous pouvez ajouter de nouveaux joueurs, faire des transferts ou encore les copier. La copie d'un joueur permet de faire une équipe nationale depuis les joueurs des clubs.

Sélectionner / désélectionner un joueur

Appuyer sur le bouton au dessus de sa petite tête.

Comment la mémoire tampon travaille

Quand vous sélectionnez un joueur, il est copié dans une mémoire tampon.

Quand cette mémoire contient un joueur, le bouton "NOUVEAU" génère un joueur copiant ses caractéristiques.

Pour effacer la mémoire tampon, désélectionnez le joueur.

Comment ajouter un nouveau joueur depuis un joueur existant

Sélectionnez un joueur et appuyez sur le bouton "NOUVEAU" pour dupliquer le joueur. C'est la manière la plus rapide d'ajouter un joueur, au lieu de partir de zéro.

Comment transférer un joueur

Sélectionnez un joueur dans son club, appuyer sur "SUPPRIMER" puis "SAUVEGARDER". Après, allez sur l'équipe de destination, appuyez sur "NOUVEAU", puis "SAUVEGARDER".

Comment ajouter un joueur à l'équipe nationale

Sélectionnez un joueur dans son club et appuyez sur "SORTIE". Allez dans l'équipe nationale et appuyez sur "NOUVEAU" puis "SAUVEGARDER"
